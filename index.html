As Tecnologias que Trabalho

Portfólio

HTML

HTML é a linguagem padrão de marcação utilizada para criar e estruturar conteúdo na web. Ele define a estrutura básica de páginas web, permitindo a inserção de texto, imagens, links e outros elementos multimídia.

CSS

CSS é uma linguagem de estilo usada para controlar a aparência visual de páginas web. Ela permite personalizar a apresentação de elementos HTML, incluindo layout, cores, fontes e responsividade, proporcionando um design consistente e atraente.

Javascript

JavaScript é uma linguagem de programação de script que permite adicionar interatividade e funcionalidades dinâmicas a páginas web. Com JavaScript, é possível criar animações, manipular elementos da DOM, validar formulários e se comunicar com servidores, melhorando a experiência do usuário.

Wordpress

WordPress é um sistema de gerenciamento de conteúdo (CMS) amplamente utilizado para criar e gerenciar sites e blogs. Ele oferece uma interface amigável e uma vasta gama de plugins e temas, permitindo a personalização e a fácil criação de sites responsivos e escaláveis.
